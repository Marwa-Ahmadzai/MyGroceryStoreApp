# 🛒 MyGroceryStoreApp

A complete Android grocery store management application built with **Jetpack Compose**, **Firebase Firestore**, and **Material 3**. This app allows users to manage grocery products with real-time updates, image uploads, and secure authentication.

---

## 📱 Features

| Feature | Description |
|:---|:---|
| 🔐 **User Authentication** | Secure login/logout with Firebase Authentication |
| 📦 **Product Management** | Add, edit, and delete products in real-time |
| 🖼️ **Image Upload** | Select images from gallery and store locally |
| 🔄 **Real-Time Updates** | Products sync instantly across all devices with Firestore listeners |
| 🎨 **Modern UI** | Built with Jetpack Compose and Material 3 design |
| 💾 **Offline Image Storage** | Images saved to internal storage for fast loading |

---

## 🛠️ Tech Stack

| Category | Technology |
|:---|:---|
| **Language** | Kotlin |
| **UI Framework** | Jetpack Compose (Material 3) |
| **Database** | Firebase Firestore (Realtime NoSQL) |
| **Authentication** | Firebase Authentication |
| **Image Loading** | Coil |
| **Architecture** | MVVM (Model-View-ViewModel) |
| **State Management** | StateFlow, MutableState |
| **Build Tool** | Gradle (Kotlin DSL) |
